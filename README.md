# git-kill

A script providing a `git kill` command which purges a file from the history of
a git repository.

## Installation

Put the script into your `$PATH`, e.g. `~/.local/bin`.

## Usage

Invoke the script via `~$ git kill <file>`. The given file will be removed from
the history.
